# Abonat

#### This is the front-end of Abonat written in Vue.js.

### Uses:
- Vuexy template - [demo](https://pixinvent.com/demo/vuexy-vuejs-admin-dashboard-template/demo-1/)
- 

### Prerequisites

- Node v16.8.0+

### Initial set up

- Clone the project
> git clone https://github.com/Kristiyan96/abonat-front.git

- Enter folder
> cd abonat-front

- Install packages
> npm i

### Run project

> npm run serve
